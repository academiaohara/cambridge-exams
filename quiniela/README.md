# Quiniela Avilés

Aplicación de quiniela y porra del Real Avilés (prototipo frontend).

## Ejecución local

```bash
cd quiniela
python3 -m http.server 8080
```

Abre http://localhost:8080/quiniela/ (o el puerto configurado).

## Secciones

- **Quiniela** — Mi quiniela a ancho completo, guardar/editar, estadísticas de votos, previas.
- **Porra del Avilés** — Marcador con casillas 0 / 1 / 2 / M y un solo goleador (o «Nadie mete»).
- **Resultado jornada** — Clasificación por jornada con barra deslizante.
- **Resultados y ranking** — Tres columnas: 1·X·2, porra (cuadrícula), goleador (balón).
