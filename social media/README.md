# 📱 Social Media Vocabulary Videos — Content Plan

## 🎯 Goal

Produce short-form vocabulary videos (Reels, TikToks, Shorts) using words from `Vocabulary.csv`.  
Each video covers **4–6 thematically related words** with **example sentences of 10–15 words** featuring celebrities, athletes, actors, or fictional characters to maximise engagement.

---

## 📐 Video Format

| Element | Details |
|---|---|
| **Words per video** | 4–6 words, same topic |
| **Example length** | 10–15 words per sentence |
| **Example subjects** | Real celebrities, athletes, actors, fictional characters |
| **Level shown** | Always show CEFR level (B1 / B2 / C1) |
| **Language** | 100% English |
| **Suggested duration** | 30–60 seconds |

### Slide structure per word
1. **Word** (large, centred)
2. **CEFR level** badge (e.g. B2)
3. **Short meaning** (one line)
4. **Example sentence** with the word highlighted
